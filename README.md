# MQTT Catalog

A comprehensive MQTT topic catalog system that collects, stores, and displays MQTT topics from multiple brokers.

## Features

- Multi-broker support (26+ brokers simultaneously)
- Automatic payload type detection (JSON, XML, Text, Binary)
- SQLite and PostgreSQL support
- RESTful API for topic retrieval
- Comprehensive unit tests (85%+ coverage)
- Stateless collector design
- Graceful shutdown handling
